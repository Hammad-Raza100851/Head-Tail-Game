# Head-Tail-Game
Head-Tail Game for beginners
Features

Player vs Computer coin toss game.

Real-time score tracking.

Option to exit the game anytime by pressing E.

Declares the final winner or a tie.

Input your choice:

h → Head

t → Tail

e → Exit the game

The computer randomly chooses Head or Tail.

The result of each toss is displayed along with the current winner.

After exiting, the final scores are displayed and the overall winner is announced.
